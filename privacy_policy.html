01

Sobre o DriverFlow

O DriverFlow é um aplicativo para motoristas de aplicativo (Uber, 99 e similares) que calcula automaticamente o lucro real de cada corrida, considerando custos de combustível, depreciação e custos fixos do veículo.

Levamos a privacidade dos nossos usuários a sério. Esta política explica de forma clara e objetiva quais dados coletamos, por que coletamos e como você pode exercer seus direitos.

02

Dados coletados

O DriverFlow coleta apenas os dados estritamente necessários para o funcionamento do app.

Dados de corridas

Valor da corrida, distância, tempo de duração e plataforma (Uber ou 99). Coletados automaticamente pela detecção de tela ou inseridos manualmente por você.

Armazenado localmente
Configuração do veículo

Tipo de veículo, consumo médio, preço do combustível e custos fixos mensais (aluguel, seguro). Inseridos manualmente por você.

Armazenado localmente
Gastos manuais

Despesas inseridas por você (combustível, alimentação, manutenção etc.) para compor os relatórios financeiros.

Armazenado localmente
Dados de diagnóstico

Logs de erro, modelo do dispositivo, versão do Android e versão do app. Coletados automaticamente pelo Firebase Crashlytics apenas quando ocorre uma falha.

Enviado ao Firebase
Identificador anônimo de dispositivo (UID)

Um identificador único gerado automaticamente pelo Firebase Authentication ao instalar o app. Não está vinculado a nenhuma identidade pessoal. Usado para funcionalidades de backup e sistema de indicação futuro.

Firebase Auth

Nunca coletamos: nome completo, CPF, RG, e-mail, senha, localização GPS em tempo real, histórico de navegação, contatos, fotos, dados bancários ou de cartão de crédito.

03

Como os dados são usados

Cálculo de lucro em tempo real

Os dados de corridas e configuração do veículo são usados exclusivamente para calcular e exibir o lucro real de cada corrida no overlay e nos relatórios.

Geração de relatórios

Os dados são processados localmente no seu dispositivo para gerar relatórios semanais e mensais com insights financeiros.

Correção de erros

Os dados de diagnóstico são usados exclusivamente para identificar e corrigir bugs do app. Nunca são usados para publicidade ou vendidos a terceiros.

Não compartilhamos seus dados com anunciantes, corretoras, plataformas de aplicativo (Uber, 99) ou qualquer outro terceiro além do Firebase Crashlytics para fins de diagnóstico técnico.

04

Serviço de Acessibilidade

O DriverFlow utiliza a API de Acessibilidade do Android (AccessibilityService) para detectar automaticamente corridas nos apps Uber Driver e 99 Motorista enquanto você trabalha.

O serviço de acessibilidade lê exclusivamente o texto visível na tela dos apps Uber Driver e 99 Motorista — especificamente o valor da corrida, distância e tempo de deslocamento exibidos no card de solicitação.

  • O serviço não lê conteúdo de outros apps, mensagens, e-mails ou qualquer outra tela.
  • O serviço não captura senhas, dados de login ou informações pessoais dos passageiros.
  • O serviço não transmite o conteúdo da tela para servidores externos — o processamento é feito inteiramente no seu dispositivo.
  • Os dados detectados ficam armazenados apenas localmente no seu dispositivo.

Você pode desativar o serviço de acessibilidade a qualquer momento em Configurações do Android → Acessibilidade → DriverFlow.

05

Firebase e serviços de terceiros

O DriverFlow utiliza os seguintes serviços do Google Firebase:

Firebase Crashlytics

Coleta relatórios de falhas automáticos quando o app apresenta erro. Inclui modelo do dispositivo, versão do Android, versão do app e stack trace do erro. Não inclui dados pessoais ou de corridas.

Google Firebase
Firebase Analytics

Registra eventos anônimos de uso do app (ex: corrida aceita, corrida rejeitada) para entender como o app é usado e melhorá-lo. Não vincula eventos a identidades pessoais.

Google Firebase
Firebase Authentication (anônimo)

O DriverFlow usa autenticação anônima do Firebase para gerar um identificador único por dispositivo (UID). Este UID não está vinculado a nenhuma identidade pessoal — não requer e-mail, senha ou qualquer dado pessoal. É usado exclusivamente para funcionalidades futuras de backup de dados e sistema de indicação entre usuários.

Google Firebase
Google AdMob (plano gratuito)

Usuários do plano gratuito podem ver anúncios exibidos pelo Google AdMob. O AdMob pode usar identificadores de publicidade do dispositivo (GAID) para exibir anúncios relevantes. Você pode redefinir ou limitar o uso desse identificador nas configurações do Android em Configurações → Google → Anúncios. Usuários do plano Premium não veem anúncios.

Google AdMob

Os serviços Firebase e AdMob são operados pela Google LLC e estão sujeitos à Política de Privacidade da Google.

06

Armazenamento e retenção

  • Dados locais (corridas, gastos, configuração do veículo) são armazenados exclusivamente no banco de dados local do seu dispositivo. Esses dados permanecem no dispositivo até você desinstalar o app ou limpar os dados do app nas configurações do Android.
  • Dados de diagnóstico enviados ao Firebase são retidos conforme a política de retenção do Firebase Crashlytics (90 dias por padrão).
  • Não mantemos servidores próprios com cópias dos seus dados de corridas.

Para excluir todos os seus dados locais: acesse Configurações do Android → Apps → DriverFlow → Armazenamento → Limpar dados, ou desinstale o app.

07

Seus direitos

Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), você tem os seguintes direitos:

  • Acesso: consultar quais dados o app armazena sobre você.
  • Retificação: corrigir dados incorretos diretamente nas telas de configuração do app.
  • Exclusão: solicitar a exclusão dos seus dados a qualquer momento. Para dados locais, basta limpar os dados do app ou desinstalar. Para dados de diagnóstico no Firebase, entre em contato conosco.
  • Portabilidade: exportar seus dados de corridas e relatórios via a função de exportação CSV disponível nos relatórios semanais e mensais.
  • Revogação de consentimento: desativar o serviço de acessibilidade a qualquer momento nas configurações do Android.
08

Segurança

Adotamos as seguintes medidas para proteger seus dados:

  • Todos os dados de corridas e configurações são armazenados em banco de dados local criptografado pelo sistema Android.
  • O app não realiza transmissões desnecessárias de dados para servidores externos.
  • O serviço de acessibilidade opera exclusivamente no contexto dos apps Uber Driver e 99 Motorista.
  • Não armazenamos senhas ou credenciais das plataformas de aplicativo.
09

Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Quando isso ocorrer, atualizaremos a data de "última atualização" no topo desta página e, se as mudanças forem significativas, notificaremos você através de um aviso dentro do app.

Recomendamos que você revise esta política periodicamente. O uso continuado do app após as alterações constitui aceitação da política atualizada.
